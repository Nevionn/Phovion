## Структура проекта

```js
SORT-PHOTOS/
├── prisma/
│   ├── dev.db
│   └── schema.prisma
│
├── public/
│   └── uploads/ - Хранилище добавленных изображений
│
├── src/
│   ├── app/
│   │   ├── album\[id]/
│   │   │   └── page.tsx - Страница выбранного альбома
│   │   └── api/
│   │       ├── albums/
│   │       │   ├── [id]/
│   │       │   │   └── route.ts - Эндпоинт для получения и удаления выбранного альбома и всех его фотографий, методы GET & DELETE
│   │       │   ├── create/
│   │       │   │   └── route.ts - Эндпоинт для создания нового альбома, метод POST
│   │       │   ├── deleteAll/
│   │       │   │   └── route.ts - Эндпоинт для удаления всех альбомов и их содержимого, метод DELETE
│   │       │   ├── reorder/
│   │       │   │   └── route.ts - Эндпоинт для сортировки альбомов путём dnd, метод POST
│   │       │   └── take/
│   │       │       └── route.ts - Эндпоинт для получения всех альбомов на главной странице(в качестве обложки устанавливается ласт картинка), метод GET
│   │       └── photos/
│   │           └── upload/
│   │               └── route.ts - Эндпоинт для загрузки фотографий в локальную папку, и созадние записей о ней в бд
├── favicon.ico
├── globals.css
├── layout.tsx
└── page.tsx - Основная страница

```
